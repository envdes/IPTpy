iptpy.anthro_emission.fv.FV#

class iptpy.anthro_emission.fv.FV(preregrid_path, regridded_path, regridder_filename, source, version, start_year, end_year, start_month=None, end_month=None, download_method=None, original_resolution=None, target_resolution='0.9x1.25', cdate=None, model_var_list=None)#

Anthropogenic emissions processing for the FV dycore in CESM.

FV class for processing global anthropogenic emissions for the FV dycore in CESM.

Parameters:

  • preregrid_path (str) – Path to pre-regridded data.

  • regridded_path (str) – Path to regridded data.

  • regridder_filename (str) – Filename to save the regridder object.

  • source (str) – Data source.

  • version (str) – Anthropogenic emission data version, either is ‘v5.3’ or ‘v6.2’ for CAMS or ‘v2021-04-21’ for CEDS.

  • start_year (int) – Starting year for processing.

  • end_year (int) – Ending year for processing.

  • start_month (int, optional) – Starting month for processing, default is 1.

  • end_month (int, optional) – Ending month for processing, default is 12.

  • download_method (str, optional) – Method to download data, either ‘wget’ or ‘globus’, default is ‘wget’.

  • original_resolution (str, optional) – Original resolution of the data, either ‘0.1x0.1’ for CAMS or ‘0.5x0.5’ for CEDS.

  • target_resolution (str, optional) – Target resolution for output, default is ‘0.9x1.25’.

  • cdate (str, optional) – Creation date for output files, default is current date.

  • model_var_list (list, optional) – List of model variables, default is all variables.
